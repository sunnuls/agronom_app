# Политика конфиденциальности — АгроСчет

**Дата:** 9 апреля 2026 г.

Приложение «АгроСчет» (Agronom) предназначено для расчёта заправки опрыскивателя и ведения профилей на устройстве пользователя.

## Какие данные обрабатываются

- Данные профилей, журнала расчётов и настроек **хранятся только на вашем устройстве** (локальная база данных приложения).
- Разработчик **не получает**, **не передаёт третьим лицам** и **не хранит на своих серверах** эти данные — **серверов приложения нет**.

## Доступ к данным

- Доступ к информации есть только у владельца устройства.
- Экспорт и обмен файлами выполняются **по вашему действию** (поделиться, сохранить) через стандартные функции системы.

## Изменения

Текст политики может обновляться; актуальная версия публикуется по этой ссылке.

## Контакт

Для связи с разработчиком используйте **E-mail для обратной связи**, указанный в разделе TestFlight / информация о приложении в App Store Connect.
